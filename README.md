# Smart Feedback Management System

A full-stack web application developed for educational institutions to automate the student feedback process.

## Features

- Student, Faculty & Admin Login
- Role-Based Authentication
- Student Feedback Submission
- Faculty Dashboard
- Admin Dashboard
- Subject Management
- Contact Form
- Responsive UI

## Technologies Used

- Java
- JSP
- Servlets
- MySQL
- JDBC
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Apache Tomcat

## Project Status

🚧 Currently under active development.

More modules including OTP Authentication, Attendance Management, Reports and Analytics will be uploaded gradually.
